DESIGN AND ANALYSIS OF ALGORITHMS

ABOUT COURSE

What is the course about?

The Design and Analysis of Algorithms course provides a comprehensive foundation for developing efficient solutions to computational problems. It introduces key algorithm design strategies like divide and conquer, dynamic programming, and greedy algorithms. It focuses on analyzing algorithm efficiency using asymptotic notations such as Big O, Big Theta, and Big Omega. The course also explores data structures like graphs, trees, and heaps for optimizing performance. Students gain skills to design and evaluate algorithms for real-world applications.

What kind of data structures and algorithms have you studied?

-Throughout this course I've learnt about Data Structures like Graphs and Trees, Tries, AVL Trees, 2-3 Trees, Red-Black Trees, DFS, BFS, Heap, Fenwick Trees, Skip Lists.

-Sorting Techniques - Bubble sort, Selection sort, Merge sort, Quick sort, Heap sort, Insertion sort

-Searching Techniques

-Graph Algorithms - Shortest Path (Floyd's Algorithm, Warshall's Algorithm), Union-find and Spanning Trees (minimun spanning trees - Kruskal's Algorithm, Prim's Algorithm), Bellman Ford, Dijkstras

-Pattern Matching - Knuth-Morris-Pratt (KMP) Algorithm, Boyer-Moore Algorithm, Rabin-Karp Algorithm
